The Data Manager page's Connections tab (aka *Connection Manager*) is used to create Data Connections, display their characteristics, and keep track of them.

![Data Manager page, Connections tab](/img/data-manager-page-connections-tab.png "Data Manager page, Connections tab")

## Data connection action menu

![Data Manager page, Connections tab, Connection menu](/img/data-manager-page-file-manager-tab-connection-menu.png "Data Manager page, Connections tab, Connection menu")

The **Action** menu on a Data Connection allows you to do the following things:

|Data Connection Action menu item |Description  |
--- | ---
|**Run**|Direct the Data Connection to immediately upload the associated file.|
|**Test**|Test that the connection works.|
|**Clone**|Create a copy of the Data Connection. You can change any of the settings during the creation process. If you clone a Data Connection another user has created, you must re-enter the password value.|
|**Delete**|Delete the connection (you can do this only if you created the connection).|

For more information on Data Connections and how to use them, see [Uploading and choosing a data file via a Data Connection](uploading-and-choosing-a-data-file-via-a-data-connection).
