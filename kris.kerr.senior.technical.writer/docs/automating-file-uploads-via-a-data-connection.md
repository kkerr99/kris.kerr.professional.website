After creating a Data Connection you can manually initiate a file upload. You can also automate the process so that the file is uploaded at certain intervals.

## To automate file uploads via a Data Connection

1. On the **Project Configuration** page in the **Automation Run Settings** section, click the **Edit** link.
2. Select the **Automate Runs** box and set the automation criteria.
    ![Configure Automation Run Settings section, Automate Runs box selected](/img/configure-automation-run-settings-page-data-connection-not-added.png "Configure Automation Run Settings section, Automate Runs box selected")
3. Click **Add Connection**.
4. In the Data Manager, create a new Data Connection and then select it from the list, or select a previously-created Data Connection.
5. Information about the selected Data Connection is displayed on the **Automation Run Settings** page.
    ![Configure Automation Run Settings section, information about the selected Data Connection is displayed](/img/configure-automation-run-settings-page-data-connection-added.png "Configure Automation Run Settings section, information about the selected Data Connection is displayed")
6. At the top right of the page, click **Save** to save your file choice.
7. At the top left of the page, click **Back**.
8. On the **Project Configuration** page, the **Automation Run Settings** section now displays the Data Connection file you just selected, along with the associated automation criteria. If the section indicator light is green, the selected data file(s) are valid.
    ![The Automation Run Settings section now displays the Data Connection file you just selected](/img/automation-run-settings-section-data-connection-added.png "The Automation Run Settings section now displays the Data Connection file you just selected")
