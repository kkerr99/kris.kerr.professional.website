The categories of users in rgOne are:

## User

A general rgOne user with normal permissions.

## App admin

This user type has the ability to:

- Do anything a general user can do. 
- Log in to the rgOne user/app admin site.
- On the rgOne user/app admin site, access an app's **User Management** page.
- On an app's **User Management** page, do user administration tasks such as:
    - Create new user accounts.
    - Create roles.
    - Assign roles to users.

## System admin

This user type has the ability to:

- Log in to the rgOne system admin site.
- On the rgOne system admin site:
    - Enter the rgOne license number that determines which rgOne apps are accessible.
    - Enable/disable and configure the installed clients.
