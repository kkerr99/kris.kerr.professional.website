## To choose input feature(s) for most model types

1. On the **Set Inputs/Outputs** page, click the **Inputs** tab.
    ![Set Inputs/Outputs page, Inputs tab](/img/inputs-outputs-section-set-inputs-page-inputs-tab.png "Set Inputs/Outputs page, Inputs tab")
2. Click **Manage Columns**. A table of the file's columns is displayed. Click the radio button or checkbox next to the column(s) you want to designate as input features.
    ![Table of the file's columns](/img/inputs-outputs-section-set-inputs-page-inputs-selected.png "Table of the file's columns")
3. Click **Done**. The newly-selected input features are displayed, along with additional information.
    ![Newly-selected input features](/img/inputs-outputs-section-set-inputs-page-inputs-selected-displayed-in-table.png "Newly-selected input features")
4. Click **Save**, then click **Back**.
5. On the **Project Configuration** page, the **Inputs/Outputs** section displays the selected input and output features.
    ![Inputs/Outputs section displays the selected input and output features](/img/configure-project-page-inputs-outputs-section-inputs-outputs-selected.png "Inputs/Outputs section displays the selected input and output features")

    The section indicator light is now green, which means the section settings are valid.
