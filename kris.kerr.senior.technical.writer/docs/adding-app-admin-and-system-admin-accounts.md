---
title: Adding app admin and system admin accounts
description: How to add an app admin or system admin account via the API
keywords:
    - API
    - admin
    - system
---

When an instance of an rgOne app is deployed, it has only two associated user accounts:

- An app admin user with the username `admin`.
- A system admin user with the username `system`.

Additional app admin and system admin users can be created only through the rgOne API.
