On the **Project Configuration** page, every section displays a status light of a given color.

![Project Configuration page section status light](/img/configure-project-page-section-indicator-light.png "Project Configuration page section status light")

| Section status light color | Meaning |
| ----------- | ----------- |
| Grey      | A user needs to set the section settings. |
| Green   | All the section settings are valid.|
| Red   | One or more section settings is invalid. |

If a section light is red, edit the section so that all the settings are valid (the section light turns green).
