title: File manager tab
description: Describes the UI of the Data Manager page's File Manager tab.

---

The File Manager tab (aka *File Manager*) is used to upload files to AML, display their characteristics, and keep track of them once uploaded.

![Data Manager page, File Manager tab](/img/data-manager-page-file-manager-tab.png "Data Manager page, File Manager tab")

## Notes

- If you upload a file through the Project Manager, a modified version of the File Manager is displayed.
- You can only delete files that you have uploaded.
- When you view a file's summary data, up to the first 200 records are displayed.
- File size values are in bytes.
