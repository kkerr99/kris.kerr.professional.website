## To add a widget to the Widget dashboard

1. Navigate to the Widget dashboard.
2. Click one of the **Add Widget** icons.
3. On the **Add Widget** popup dialog, click one of the **Add Widget** buttons.
![Add Widget dialog](/img/add-widget-dialog.png "Add Widget dialog")
4. On the widget **Configuration** popup dialog, enter appropriate configuration values and select appropriate options.
![Widget Configuration dialog](/img/chart-configuration-dialog.png "Widget Configuration dialog")
5. Click **OK**.
6. The widget is added to the dashboard.
