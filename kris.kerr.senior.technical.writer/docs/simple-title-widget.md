---

Use the Title widget to create page or sectional titles and labels using a variety of fonts, sizes, and color schemes.

## Simple Title widget selection dialog

![Simple Title widget selection dialog](/img/simple-title-widget.png "Simple Title widget selection dialog")

## Simple Title widget configuration dialog

![Simple Title Configuration dialog](/img/simple-title-configuration.png "Simple Title Configuration dialog")

### Title
The text displayed in the widget.

### Font Family
The font to be applied to the title text.

### Font Size
The size of the font to be applied to the title text.

### Font Color
The color to be applied to the title text. Note that there are multiple ways to define the color (hex value, keyword, and so on).
