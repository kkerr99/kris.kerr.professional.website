## Overview

Anything that you can do through the rgOne web apps, you can do by calling the rgOne API endpoints. Furthermore, rgOne API endpoints include functionality not included in the UI.

## rgOne API documentation URL

We use the open-source [Swagger](https://swagger.io/) toolset to automatically generate the rgOne API documentation.
