## To add a user role

1. Log on to rgOne as an app admin.
2. On the left vertical navigation bar, click an rgOne app icon and click an app page. This action determines the selected app.
3. On the left vertical navigation bar, click the **User** icon:  
    ![Navigation bar User icon](/img/model-manager-page-user-icon.png "Navigation bar User icon")

    On the popup menu, select **User Management**.  
    ![User menu, User Management menu item](/img/user-menu-user-management.png "User menu, User Management menu item")

    The **User Management** page is displayed.  
    ![User Management page](/img/aml-user-management-users-page.png "User Management page")
4. On the top of the page, click the **Roles** tab. The **Roles Management** page is displayed, and all the existing roles for the selected app are listed.
    ![Role Management page](/img/aml-user-management-roles-page.png "Role Management page")
5. Click **Add Role**. The **Add Role** dialog is displayed.
    ![Add Role dialog](/img/aml-user-management-add-role-dialog.png "Add Role dialog")
6. Enter a **role name** and **description**.
7. Select one or more of the possible permissions from the list:
    - Select **Active Permissions** to display only the permissions youâ€™ve already selected.
    - You can also filter the permissions displayed on this list by entering text into the **Filter Permissions** box. For example, if you enter `publish`, only the permissions with the text `publish` in the name or description are displayed.
8. Click **Add Role**.
